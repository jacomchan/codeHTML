What are html tags? HTML tags are the keywords on a web page that define how your web browser must format and display your web page. they start <> and add a / inside it to end it.

    Main HTML Tags:

    Tag Start Tag End Tag Description
    html Yes Yes Represents the root of an HTML document.
    head Yes Yes Defines the head section of an HTML document.
    body Yes Yes Define the body of the HTML document.
    h1 - h6 Yes Yes Define HTML headings.
    ol Yes Yes Defines an ordered list.
    li Yes Yes Used to represent an item in a list.
    p Yes Yes Defines a paragraph.
    img Yes No Used to incorporate in-line graphics (typically icons or small graphics) into an HTML document. scr="" (To link the file) alt="" (The description of the image)
